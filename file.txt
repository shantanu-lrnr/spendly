❯ what is this project about                                                    

  Searched for 3 patterns, read 2 files (ctrl+o to expand)

● This is a Flask-based expense tracker web app — a teaching/learning project
  with a step-by-step implementation guide for students.

  Current state:
  - Flask app with routes for landing, register, login, logout, profile, and
  CRUD expense operations
  - Most routes are placeholders with "coming in Step X" messages
  - Templates exist for landing, login, register, and a base layout
  - A SQLite database module (database/db.py) is stubbed out but not yet
  implemented

  Planned features (based on the step structure):
  1. Database setup (SQLite with get_db, init_db, seed_db)
  2. Landing page
  3. Auth: register/login/logout
  4. User profile
  5-9. Expense management: add, edit, delete expenses

  It's essentially a starter scaffold for a personal finance tracking app,
  intended for students to fill in incrementally.
